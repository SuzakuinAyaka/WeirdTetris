# 更新日志

本文件记录项目的所有重要变更。

## [1.2.3] - 2026-04-02

### 新增

- 教程补充“双击下移可直接沉底”说明（中英文与中文资源同步）。

### 变更

- 首页语言卡片标题固定为英文 Language，描述继续随当前语言显示“切换应用语言”对应文案。
- 首页“商店 / 背包”入口改为常驻显示，不再受道具模式开关影响。
- 首页语言与设置卡片高度统一，视觉对齐更稳定。
- 设置页与教程页改为“固定标题 + 下方滚动内容”结构，标题不再随内容滚动。
- 小版本迭代：1.2.2 -> 1.2.3（versionCode: 10202 -> 10203）。

### 验证

- 执行 :app:compileDebugJavaWithJavac 与 testDebugUnitTest 进行编译与单元测试验证。

## [1.2.2] - 2026-04-01

### 新增

- 游戏页下移按钮新增双击直落能力：短时间内连续点击两次下移按钮，可让当前方块直接沉底并锁定。

### 修复

- 修复游戏页快捷栏道具按钮显示问题：快捷栏按钮统一只显示道具名称，数量仅在上方计数按钮显示（薯条等道具一致）。

### 变更

- 调整平板横屏游戏页底部操作栏顺序为：左移 -> 旋转 -> 下移 -> 右移。
- 拉开平板横屏底部操作栏中“旋转”与“下移”按钮间距，降低误触。
- 小版本迭代：`1.2.1` -> `1.2.2`（`versionCode: 10201 -> 10202`）。

### 验证

- 完成代码快速 review，并执行 `:app:compileDebugJavaWithJavac` 与 `testDebugUnitTest` 验证。

## [1.2.1] - 2026-03-31

### 修复

- 修复中文环境下旋转暂停弹窗文案显示为英文的问题，补齐 `values-zh` 的 `rotation_pause_title`、`rotation_pause_message`、`resume_game` 资源。

### 变更

- 将 `CHANGELOG.md` 全量调整为中文描述，统一文档语言风格。

## [1.2.0] - 2026-03-31

### 修复

- 修复平板横屏下状态栏遮挡内容的问题：在 `DrawerLayout` 场景中正确分发系统栏 `insets`。
- 修复屏幕旋转后对局继续运行的问题：旋转恢复后会先暂停并提示，再由用户确认继续。

### 变更

- 平板横屏模式下，移动/旋转/下落操作按钮统一调整到底部操作栏。
- 旋转暂停弹窗新增背景模糊效果（Android 12+）。
- 版本升级：`1.1.0` -> `1.2.0`（`versionCode: 10100 -> 10200`）。

## [1.1.0] - 2026-03-30

### 修复

- Gradle Wrapper 分发地址切换为腾讯镜像 `mirrors.cloud.tencent.com/gradle`，并使用 `all` 包。
- 修复动态配色开关不能即时生效的问题。
- 修复全局 `WindowInsets` 仅处理顶部导致底部控件被遮挡的问题。
- 修复平板旋转后对局状态丢失问题，`GameActivity` 新增对局状态保存与恢复。

### 变更

- `settings.gradle.kts` 调整为腾讯镜像优先，官方仓库作为兜底。
- 版本升级：`1.0.0` -> `1.1.0`（`versionCode: 10000 -> 10100`）。
- README 结构重构，并将更新记录独立到本文件。

### 新增

- 新增单元测试：`GameConstantsTest`、`TetrominoBagGeneratorTest`。

## [1.0.0] - 2026-03-29

### 新增

- 首个版本发布：包含核心玩法（无尽/挑战）、道具与金币系统、商店/背包、设置/教程，以及手机与平板适配。
